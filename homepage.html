About HolDex

The High-Velocity Terminal for Solana Assets

Core Philosophy

HolDex is built for traders who demand raw speed and absolute truth. In an ecosystem flooded with noise, we provide a terminal-grade interface that cuts through the clutter.

We prioritize community verification and on-chain data integrity. Our proprietary indexing engine snapshots liquidity pools directly from the Solana RPC, ensuring you see price action as it happens, not minutes later.

No third-party delays. We index Raydium and Pump.fun directly.

HolDex API Reference

Base URL: https://asdev-backend.onrender.com

GET /api/tokens

Fetch a paginated list of tokens indexed by the protocol.

// Example Request fetch('https://asdev-backend.onrender.com/api/tokens?sort=kscore&page=1'); // Query Parameters ?sort -> 'kscore', 'newest', 'volume', 'mcap', '24h' (default: kscore) ?page -> Integer (default: 1) ?limit -> Integer (default: 100) ?search -> String (Ticker, Name, or Mint Address)
GET /api/token/:mint

Get detailed metadata, price stats, and pool information for a specific token.

// Example Request fetch('https://asdev-backend.onrender.com/api/token/So111...11112');
GET /api/token/:mint/candles

Fetch OHLCV candle data for charting. Returns data from the most liquid pool by default.

// Example Request fetch('.../api/token/:mint/candles?resolution=60&from=1700000000&to=1700086400'); // Query Parameters ?resolution -> '5', '15', '60', '240', 'D' (Minutes, default: 5) ?from -> Unix Timestamp (Seconds) ?to -> Unix Timestamp (Seconds)
